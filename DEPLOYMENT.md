# Deployment Notes

## Option 1: GitHub Pages website

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Upload `index.html`, `style.css`, `script.js`, and your CV PDF.
3. Rename your CV PDF to `CV-Dr-Muhammad-Reshaeel.pdf`, or update the CV link in `index.html`.
4. In GitHub, open **Settings > Pages** and publish from the `main` branch.
5. Your website will be available at `https://<your-github-username>.github.io/`.

## Option 2: GitHub profile README

1. Create a repository named exactly the same as your GitHub username.
2. Add the `README.md` file from this folder.
3. GitHub will display it automatically at the top of your profile.
